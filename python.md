## What Python *code* style do I use?
We follow the [Style Guide for Python Code](https://www.python.org/dev/peps/pep-0008/).

---

### Naming Style

- Variables, functions, methods, modules, packages: `lower_case_with_underscores`
- Constants: `ALL_CAPS_WITH_UNDERSCORES`
- Classes and exceptions: `CapWords`

### Indentation

- 4 spaces or 1 tab

### Space

- Leave space around operators:

```python
var = 4
```
- Leave space after commas:
```python  
var_1, var_2 = get_values(num_1, num_2)
```
### Comments
- Comments are introduced with `#`, placed above the relevant code, and aligned with the code.
```python 
# assign variable
a = 5 
```
*Why are comments important?*
As part of a community, we want code to be readable and reusable by others. Be generous and precise with comments.
