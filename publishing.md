## In what journal can I publish my Python package?
Consider publishing in the [Journal of Open Source Software (JOSS)](https://joss.theoj.org/). The paper is only a few pages and reviewers check your repository, helping you improve it.

---

## How do I get a DOI for my code?
To get a permanent Digital Object Identifier (DOI) for your code so that it can be referenced, you can link your GitHub repository to Zenodo. You can find instructions on [this webpage](https://docs.github.com/en/repositories/archiving-a-github-repository/referencing-and-citing-content) or in [this video](https://www.youtube.com/watch?v=gp3D4mf6MHQ).
